PKGO Web - V1.10
====================
本次更新內容：
1) 清單排序固定為「頂尖級 → 高階級 → 基礎級」，同級以名稱排序；未選等級者排最後。
2) 變更等級後（下拉選單），列表會即時重排。
3) 新增 Service Worker（sw.js）與註冊邏輯：PWA/桌面版一開就檢查更新並自動套用。
4) 版本號：V1.10。

如何使用：
- 將 `main.V1.10.js` 引入到你的頁面（在原本 main.js 後面引入，或直接合併到主檔）。
- 把 `sw.js` 放到網站根目錄（若是 GitHub Pages 子目錄，請確保 sw.js 與頁面位於同一根路徑下）。
- 部署時記得每次修改 `SW_VERSION`（此版本為 'V1.10'），以觸發新版快取。
- 若你的專案不是放在根路徑，請把 sw.js 內的 `STATIC_ASSETS` 與 main.V1.10.js 的 base path 調成你的子路徑（例如 `/PKGO-MEGA-Web/`）。
